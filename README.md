# HR Employee Attrition Analysis Dashboard

## Project Overview

This project analyzes employee attrition patterns using SQL and Power BI. The goal was to identify factors contributing to employee turnover and generate insights to support HR decision-making.

## Tools Used

- MySQL
- Power BI
- DAX
- Power Query

## Key Metrics

- Total Employees: 1470
- Employees Left: 237
- Attrition Rate: 16.12%
- Average Monthly Income: 6502.93

## Analysis Performed

### Department Analysis
Research & Development recorded the highest employee attrition, followed by Sales and Human Resources.

### Job Role Analysis
Sales Representatives and Laboratory Technicians experienced the highest attrition among job roles.

### Overtime Analysis
Employees working overtime showed significantly higher attrition compared to employees who did not work overtime.

### Age Group Analysis
Employees under 30 years of age had the highest attrition levels, indicating greater turnover among younger employees.

### Gender Analysis
Attrition was observed across both genders, with slightly higher employee exits among male employees.

## Dashboard Features

- KPI Cards
- Department-wise Attrition Analysis
- Job Role Attrition Analysis
- Overtime Attrition Analysis
- Age Group Attrition Analysis
- Gender Analysis
- Interactive Department Filter
- Interactive Gender Filter

## Key Insights

- Overtime is a major factor influencing employee attrition.
- Younger employees are more likely to leave the organization.
- Research & Development and Sales departments require stronger retention strategies.
- Certain job roles face considerably higher turnover rates.

## Project Outcome

Built an interactive HR analytics dashboard using SQL and Power BI to monitor employee attrition trends and support data-driven workforce decisions.
